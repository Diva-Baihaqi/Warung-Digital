## ‚ú® Fitur Utama

### üõí Halaman Pelanggan (Front-End)

- **Desain Neo-Brutalism**: Gaya desain yang berani, kontras, dan ekspresif.
- **Katalog Produk**: Lengkap dengan filter kategori dan pencarian cepat.
- **Manajemen Keranjang (Cart)**: Tambah, update, dan hapus item dengan mudah.
- **Checkout & Pembayaran**: Mendukung upload bukti pembayaran WhatsApp/Transfer.
- **Member Dashboard**: Pantau status pesanan, riwayat pembelian, dan edit profil.
- **Interaksi Pengguna**: Fitur ulasan produk dan rating.

### üõ† Panel Admin (Back-End)

- **Dashboard Statistik**: Grafik penjualan, total produk, dan pelanggan baru.
- **Manajemen Inventaris**: CRUD Produk, Kategori, dan Stok (Stock Management).
- **Manajemen Pesanan**: Verifikasi bukti bayar, update status (Pending -> Complete).
- **Laporan & Analitik**: Cetak laporan penjualan harian/bulanan (PDF/Excel).
- **Manajemen Konten**: Kelola artikel blog, FAQ, dan data sosial media.

## ‚öôÔ∏è Instalasi & Konfigurasi

Ikuti langkah-langkah berikut untuk menjalankan proyek di komputer lokal Anda:

1.  **Clone Repository**

    ```bash
    git clone https://github.com/username/warung-digital.git
    cd warung-digital
    ```

2.  **Konfigurasi Database**

    - Pastikan Anda memiliki database server (MySQL/MariaDB) aktif.
    - Buat database baru bernama `warung_digital`.
    - Import file SQL yang berada di `database/warung_digital.sql` ke dalam database yang baru dibuat.

3.  **Pengaturan Koneksi**

    - Buka file `config/database.php`.
    - Sesuaikan konfigurasi berikut dengan kredensial database Anda:
      ```php
      $host = 'localhost';
      $user = 'root';
      $pass = ''; // Sesuaikan jika ada password
      $db   = 'warung_digital';
      ```

4.  **Menjalankan Aplikasi**
    - Gunakan web server lokal seperti Apache (XAMPP/Laragon).
    - Arahkan root direktori server ke folder proyek ini.
    - Akses melalui browser: `http://localhost/warung-digital` (atau sesuai konfigurasi virtual host Anda).
